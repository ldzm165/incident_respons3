# Incident Analysis Checklist

## Phase 1: Triage (0-15 minutes)
- [ ] Review `10_Triage_Summary.txt`
- [ ] Check for critical process names
- [ ] Identify unusual network connections
- [ ] Note collection time discrepancies

## Phase 2: User Analysis (15-30 minutes)
- [ ] Review `02_User_and_Group_Information.txt`
- [ ] Identify unauthorized accounts
- [ ] Check administrator group members
- [ ] Review login sessions

## Phase 3: Persistence (30-45 minutes)
- [ ] Analyze `06_Automation_and_Scheduling.txt`
- [ ] Check registry autoruns
- [ ] Review scheduled tasks
- [ ] Examine startup folders

## Phase 4: System Changes (45-60 minutes)
- [ ] Review `09_Additional_Forensic_Artifacts.txt`
- [ ] Check recently installed software
- [ ] Review system patches
- [ ] Examine environment variables
